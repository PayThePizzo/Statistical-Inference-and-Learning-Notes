# Statistical Inference and Learning Notes

Notes and labs from the course ["Statistical Inference and Learning"](https://www.unive.it/data/insegnamento/521944) at Ca' Foscari University of Venice (for the year 2025/2026), taught by Professor [Varin Cristiano](https://www.unive.it/data/persone/5592728).

All the credits go to the professor, the creators and owners of the resources used in this course.

---
## Premises
The course is supposed to be a sort of a "practical course", even though the theoretical aspects should not be ignored. In fact, the professor remarked that he will not be able to include all the explainations due to lack of time.

This is why we built this repo: it represents an attempt to gather the essential aspects needed for the course, in order to help fellow students.

## Resources
The course's main page and the given slides specify the material used already, but we will try to cite it wherever possible.

### Referral Textbooks
The main textbook is the following
* [An Introduction to Statistical Learning by Gareth James, Daniela Witten, Trevor Hastie, Robert Tibshirani](https://www.statlearning.com/)

The professor also suggests to refresh the basic notions with this book:
* [Probability and Statistics for Computer Scientists by Michael Baron]([https://www.statlearning.com/](https://dr-baron.github.io/index.html#book))

### Latex-related resources
Shoutout to 2 crazy helpful Latex-related resources:
* [Matthias Heinkenschloss - Latex Symbols Pdf](https://www.cmor-faculty.rice.edu/~heinken/latex/symbols.pdf)
* [CODECOGS - Equation Editor](https://latex.codecogs.com/eqneditor/editor.php)

### Free access lecture notes and books
The following material should be easily accessible for the
* [David Dalpiaz - Applied Statistics with R!](https://book.stat420.org/)
* [Julian J. Faraway - Practical Regression and Anova using R](https://cran.r-project.org/doc/contrib/Faraway-PRA.pdf)
* [Cosma Shalizi - A draft textbook on data analysis methods](https://www.stat.cmu.edu/~cshalizi/ADAfaEPoV/)
* [Rafael A. Irizarry - Introduction to Data Science](http://rafalab.dfci.harvard.edu/dsbook/)
* [Hastie, Tibshirani and Friedmam - The elements of statistical learning](https://hastie.su.domains/ElemStatLearn/download.html)
